Complete installation and configuration guide for your organization

1 Clone or Download AROG

Get the AROG repository to your local machine:

git clone https://github.com/YourOrg/AROG.git
cd AROG

2 Run Automated Setup

AROG includes an automated setup script that configures everything:

npm run arog:setup

This will:

  • ‚úÖ Install all dependencies
  • ‚úÖ Install Playwright browsers
  • ‚úÖ Create directory structure
  • ‚úÖ Validate configuration

3 Verify Installation

Validate that everything is configured correctly:

npm run arog:validate
‚úÖ Expected Output:
All 10 automation types configured
8 GitHub Actions workflows ready
AROG agent installed
System fully operational

4 Connect to GitHub

Create a GitHub repository and push AROG:

# Create repo on GitHub first, then:
git remote add origin https://github.com/YourOrg/AROG.git
git branch -M main
git commit -m "feat: Add AROG automation framework"
git push -u origin main
‚ö†Ô∏è Important: Make sure to create the repository on GitHub before pushing

5 Configure GitHub Actions

Enable GitHub Actions in your repository:

  1. Go to your GitHub repository
  2. Click on "Actions" tab
  3. Click "I understand my workflows, go ahead and enable them"
  4. Workflows will now run automatically on commits and PRs

6 Activate AROG Agent in VS Code

The AROG agent is automatically available in VS Code GitHub Copilot:

  1. Open VS Code
  2. Open GitHub Copilot Chat (Cmd+Shift+I or Ctrl+Shift+I)
  3. Type: @arog review this code
  4. AROG will respond and help you!

7 Test the System

Run all tests to ensure everything works:

# Unit tests
npm test

# E2E tests  
npm run test:e2e

# Security scan
npm run security:audit

# Build check
npm run build

# All validation
npm run arog:health

8 View Documentation

Start the documentation server:

npm run docs:serve

Open your browser to: http://localhost:8080
